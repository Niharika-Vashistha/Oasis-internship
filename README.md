
## 🚀 Overview

This repository contains machine learning tasks. The work demonstrates practical implementation of regression and classification models using real-world datasets and synthetic data.

## 📌 Tasks Covered

### 1️⃣ Advertising Sales Prediction (Regression)

**Dataset:** Advertising.csv
**Objective:** Predict product sales based on advertising spend across TV, Radio, and Newspaper.

#### 🔧 Steps Performed:

* Data loading and exploration
* Checking missing values
* Feature-target separation
* Train-test split (80-20)
* Model training using Linear Regression

#### 📈 Results:

* Mean Squared Error (MSE): ~3.19
* R² Score: ~0.89

#### 📊 Visualization:

* Scatter plot comparing Actual vs Predicted Sales

---

### 2️⃣ Iris Classification (Multi-class Classification)

**Dataset:** Iris Dataset
**Objective:** Classify flowers into different species.

#### 🔧 Algorithms Used:

* Logistic Regression
* K-Nearest Neighbors (KNN)

#### 📈 Results:

* Achieved **100% accuracy** on test data
* Confusion matrix visualization included

#### ⚠️ Note:

* Logistic Regression showed convergence warning (due to iteration limit)

---

### 3️⃣ Car Price Prediction & Model Comparison

**Dataset:** Car Data.csv + Synthetic Data
**Objective:** Predict car selling prices and compare classification models.

#### 🔧 Steps Performed:

* Data preprocessing and cleaning
* Feature selection
* Handling categorical variables
* Train-test split

#### 🤖 Models Used:

* Logistic Regression (multiclass)
* Support Vector Machine (SVM)

#### 📈 Results:

* Accuracy ranged between **~53% to ~81%**
* Performance comparison across models

#### 📊 Visualization:

* Confusion matrix heatmaps
